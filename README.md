MkDocs plugin to generate a manpage from the documentation site.

## Installation

With `pip`:
```bash
pip install mkdocs-manpage
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.7 -m pip install --user pipx
pipx install mkdocs-manpage
```
